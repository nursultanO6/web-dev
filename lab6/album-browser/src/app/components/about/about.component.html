About This App

📖 About the Application

Album Browser is an Angular single-page application (SPA) built as part of Lab 6. It demonstrates key Angular concepts including routing, HTTP requests via HttpClient, Observable-based reactive data handling, and a service-layer architecture.

Data is fetched from JSONPlaceholder, a free public REST API that provides fake data for testing and prototyping. The app supports browsing 100 albums, viewing their photos, editing album titles, and deleting albums from the list.

⚙️ Technologies Used

  • Angular 16+ Frontend framework
  • TypeScript Strongly-typed JavaScript
  • Angular Router Client-side routing
  • HttpClient REST API communication
  • RxJS Reactive programming with Observables
  • JSONPlaceholder Public API

🎯 Key Features

  • Multi-route SPA with Angular Router
  • Service layer for all HTTP calls (no direct HttpClient in components)
  • CRUD operations: Read, Update, Delete
  • Loading indicators and graceful empty states
  • Responsive photo grid layout
  • Scoped component styles
